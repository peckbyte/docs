# 食材数据库

## 数据库结构



食物数据结构

| 数据名称   | 数据类型 | 说明     | 可选值                 | 默认值 |
| ---------- | -------- | -------- | ---------------------- | ------ |
| name       | string   | 食物名称 |                        |        |
| unit       | string   | 食物单位 |                        |        |
| _id        | string   | 存储单位 |                        |        |
| createTime | string   | 创立时间 |                        |        |
| type       | string   | 类型     | 肉类、蔬菜、水果、鱼类 | null   |
